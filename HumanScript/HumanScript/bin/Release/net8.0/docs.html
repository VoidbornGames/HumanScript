1. Overview

HumanScript is a simple, English-like programming language designed to be easily readable and writable by humans. It compiles to 32-bit Windows assembly and is then linked into an executable file (.exe). The language aims to abstract away complex syntax while still providing fundamental programming capabilities.

Note: HumanScript source files use the .eng extension.

2. Getting Started

2.1 Prerequisites

To compile HumanScript code, you need the following tools in the same directory as the compiler or in your system's PATH:

  • NASM (Netwide Assembler) - For assembling the generated assembly code.
  • GoLink - For linking the assembled object file into an executable.

2.2 Compiling a Program

To compile a HumanScript file, run the compiler with the following command:

HumanScript.exe [input_file.eng]

If no input file is specified, it defaults to script.eng. The compiler will produce an executable file named output.exe (or the name of the input file with a .exe extension).

2.3 Hello World Example

Here's a simple "Hello, World!" program in HumanScript:

# Define a string variable
define message as "Hello, World!".

# Print the message to the console
print message.

3. Language Features

3.1 Variables and Data Types

Variables in HumanScript are defined using the define keyword. The language supports three basic data types:

  • String: Text enclosed in double quotes.
  • Number/Integer: Whole numbers.
  • Boolean: true or false.

Syntax:

define variable_name as value.

Examples:

define name as "John Doe".
define age as 30.
define is_student as true.

3.2 Input and Output

Printing to Console

You can print either variables or literal strings to the console.

Syntax:
print variable_name.
print "Literal string".
Example:
define greeting as "Welcome".
print greeting.
print "This is a literal string."

Getting User Input

You can store user input from the console into a variable.

Syntax:
store console input in variable_name.
Example:
define user_name as "".
print "Please enter your name:".
store console input in user_name.
print "Hello, ".
print user_name.

3.3 Operations

Arithmetic Operations

HumanScript supports basic arithmetic operations: addition, subtraction, multiplication, and division.

Syntax:
# Modify a variable by a number
add number to variable.
subtract number from variable.
multiply variable by number.
divide variable by number.

# Set a variable to the result of an operation
set variable to number plus number.
set variable to variable minus number.
set variable to number times variable.
set variable to variable divided by variable.
Example:
define counter as 0.
add 5 to counter.
print counter.  # Outputs 5

set counter to counter times 2.
print counter.  # Outputs 10

String Operations

String Concatenation

You can combine two strings into a new string.

Syntax:
set destination_string to source_string1 combined with source_string2.
Example:
define first_name as "John".
define last_name as "Doe".
define full_name as "".
set full_name to first_name combined with last_name.
print full_name.  # Outputs "JohnDoe"
Number to Text Conversion

You can convert a numeric variable to its text representation.

Syntax:
turn number_variable to text as string_variable.
Example:
define age as 25.
define age_text as "".
turn age to text as age_text.
print "You are ".
print age_text.
print " years old."

3.4 Control Flow

Conditional Statements (If/Else If/Else)

HumanScript supports conditional logic with if, else if, and else. Code blocks are defined by indentation.

Syntax:
if condition:
    # Code to execute if condition is true
    [indented code]

else if another_condition:
    # Code to execute if another_condition is true
    [indented code]

else:
    # Code to execute if no conditions are true
    [indented code]
Conditions:
variable is true
variable is false
variable is equal to variable_or_number
variable is not equal to variable_or_number
variable is greater than variable_or_number
variable is less than variable_or_number
Example:
define age as 18.

if age is greater than 18:
    print "You are an adult.".
else if age is equal to 18:
    print "You just became an adult.".
else:
    print "You are a minor.".

Loops

HumanScript has a simple repeat loop to execute a block of code a specific number of times.

Syntax:
repeat number times
[
    [indented code to repeat]
]
Example:
define counter as 0.
repeat 5 times
[
    print "Loop iteration".
    add 1 to counter.
]
print counter.  # Outputs 5

3.5 Functions

You can define reusable blocks of code as functions and call them from your main program or other functions.

Defining a Function

define function named function_name
[
    [indented function body]
]

Calling a Function

run function function_name.

Example:

define function named say_hello
[
    print "Hello from a function!".
]

# Main program
print "About to call a function."
run function say_hello.
print "Function call finished."

3.6 System Interactions

Waiting

You can pause the program's execution for a specified number of seconds.

Syntax:
wait for number seconds.
Example:
print "Waiting for 3 seconds...".
wait for 3 seconds.
print "Done waiting."

Running External Processes

You can execute external programs.

Syntax:
run process "path_to_executable".
Example:
run process "notepad.exe".

File Operations

You can write text or variables to a file.

Syntax:
# Write a literal string
write "text to write" to "file_path.txt".

# Write a variable's content
write variable_name to "file_path.txt".
Example:
define log_message as "Program started successfully.".
write log_message to "C:\Temp\log.txt".
write "Another log entry." to "C:\Temp\log.txt".

4. Example Program

Here is a more comprehensive example that demonstrates several features of HumanScript:

# A simple user information collector

define user_name as "".
define user_age as 0.
define output_file as "user_info.txt".

print "--- User Information Collector ---".
print "Please enter your name:".
store console input in user_name.

print "Please enter your age:".
store console input in user_age.

# Create a summary message
define summary as "".
set summary to "Name: " combined with user_name.
set summary to summary combined with ", Age: ".
turn user_age to text as age_text.
set summary to summary combined with age_text.

# Print the summary
print "Summary:".
print summary.

# Write the summary to a file
write summary to output_file.
print "Information saved to ".
print output_file.

# Wait before exiting
wait for 2 seconds.
print "Program finished."

5. Syntax Reference

This section provides a quick reference for all HumanScript commands.

Command Syntax Description
Variable Definition define name as value. Declares and initializes a variable.
Print Variable print variable_name. Prints the value of a variable to the console.
Print String print "string". Prints a literal string to the console.
Get Input store console input in variable_name. Reads a line of input from the console into a variable.
Add To add number to variable. Adds a number to a variable's value.
Subtract From subtract number from variable. Subtracts a number from a variable's value.
Multiply By multiply variable by number. Multiplies a variable's value by a number.
Divide By divide variable by number. Divides a variable's value by a number.
Set To Operation set var to val1 plus val2. Sets a variable to the result of an arithmetic operation (plus, minus, times, divided by).
Set To Number set variable to number. Sets a variable to a specific number.
Set To Boolean set variable to true/false. Sets a boolean variable to true or false.
String Concatenation set dest to src1 combined with src2. Concatenates two strings and stores the result.
Number to Text turn num_var to text as str_var. Converts a number to its string representation.
If Statement if condition: Starts a conditional block.
Else If Statement else if condition: Starts an alternative conditional block.
Else Statement else: Starts a block for when no conditions are met.
Repeat Loop repeat number times Starts a loop that iterates a specified number of times.
End Block ] Signifies the end of an indented code block (for loops, functions, etc.).
Define Function define function named name Starts the definition of a function.
Run Function run function name. Calls a previously defined function.
Wait wait for number seconds. Pauses execution for a given number of seconds.
Run Process run process "path". Executes an external program.
Write to File write content to "path". Writes a string or variable's content to a file.
Comment # This is a comment Single-line comment. The compiler ignores everything after # on a line.
